## Controles

### Movimiento
- **W / A / S / D** — Moverse
- **Shift** — Correr

### Cámara
- **Ratón** — Rotar la cámara
- **Click derecho** — Cambiar entre **primera persona** y **tercera persona**

### Acciones
- **Click izquierdo** — Atacar
- **B** — Bailar
- **G** — Guardar partida
- **E** — Interactuar
